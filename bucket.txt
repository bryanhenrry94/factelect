| Carpeta             | Contenido                                       | Acceso recomendado          | Notas                                                 |
| ------------------- | ----------------------------------------------- | --------------------------- | ----------------------------------------------------- |
| **xml/generados**   | XML sin firmar (recién generados)               | Privado                     | Son temporales; se eliminan tras la firma             |
| **xml/firmados**    | XML ya firmados con el certificado digital      | Privado                     | Solo el backend debe poder acceder                    |
| **xml/autorizados** | XML autorizados por el SRI                      | Público o presigned URL     | Estos son válidos legalmente, puedes ofrecer descarga |
| **pdf/**            | PDF de representación gráfica (factura impresa) | Público o presigned URL     | Se puede compartir al cliente                         |
| **certificados/**   | Archivos `.p12` de cada tenant                  | **Privado (nunca público)** | Solo accesibles desde el backend                      |
| **logs/**           | Archivos o reportes de ejecución o errores      | Privado                     | Útiles para auditorías o depuración                   |
